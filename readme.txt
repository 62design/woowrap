Wraps Woocommerce in Bootstrap classes using the SASS @extend directive.

== Description ==
Woowrap is a Woocommerce extension that disables the default Woocommerce style and adds the bootstrap classes for responsiveness to the Woocommerce front-end.

To swap the sidebar and main content, just add the class "sidebar-swap" to the body.

== Installation ==
1. Ensure Woocommerce is installed and activated
2. Install and Activate Woowrap

No need to include bootstrap as woowrap contains a minified version of bootstrap.
